Fun Facts About Internet and Computers

The First Website

The first website ever created went live in 1991 and was dedicated to information about the World Wide Web project.

Internet Speed Milestones

The fastest internet speed recorded was 178 terabits per second achieved in 2020 by a team of Japanese researchers.

Computer Memory Growth

The amount of data storage available today exceeds the estimated number of stars in the observable universe.
